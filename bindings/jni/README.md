# JNI Bindings (UniFFI-generated)

UniFFI-generated Kotlin bindings for the Fawx engine.

**Status:** Stub. Will be generated once the kernel crate has a public API.

## Architecture

- Kotlin drives, Rust thinks
- One fat FFI call per loop cycle: `run_loop_cycle(perception) → LoopResult`
- Generated by Mozilla UniFFI from Rust type definitions
- Simple cargo build + copy `.so` — no Gradle plugin
