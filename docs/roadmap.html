Core Engine

Single Rust binary. Agentic loop with tool calling, memory, and multi-provider support.

  • ✓ 7-step agentic loop with multi-tool calls
  • ✓ 13+ built-in tools (files, web, browser, code execution)
  • ✓ Persistent memory with semantic search
  • ✓ Multi-provider (Claude, GPT, local models)
  • ✓ Encrypted credential store (AES-256-GCM)
  • ✓ Config management with hot-reload (SIGHUP restart)
  • ✓ Token-by-token streaming over SSE
  • ✓ Reflective memory (fx-journal crate)
  • ✓ Conversation compaction (sliding window + LLM summarization)

The critical path to replacing external AI assistants. Context compaction with memory persistence, robust error surfacing, and cross-session communication.

  • ◐ Context compaction with memory flush (compaction engine built, needs pre-compaction journal write)

  • ○ Cross-session messaging (session-to-session communication bus)
  • ○ Cron / scheduled tasks (daemon-level, persist across restarts)
  • ○ Heartbeat monitoring + self-recovery watchdog
