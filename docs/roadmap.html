Fawx Roadmap

Internal. What's built, what's in progress, and where we're headed.

Shipped

Core Engine

Single Rust binary. Agentic loop with tool calling, memory, and multi-provider support.

  • ✓ 7-step agentic loop with multi-tool calls
  • ✓ 13+ built-in tools (files, web, browser, code execution)
  • ✓ Persistent memory with semantic search
  • ✓ Multi-provider (Claude, GPT, local models)
  • ✓ Encrypted credential store (AES-256-GCM)
  • ✓ Config management with hot-reload (SIGHUP restart)
  • ✓ Token-by-token streaming over SSE
  • ✓ Reflective memory (fx-journal crate)

Kernel Safety Architecture

Hard kernel/loadable separation. Compiled safety invariants that the agent cannot modify.

  • ✓ Proposal gates (human approval for writes)
  • ✓ Tier-3 immutable paths (compiled into binary)
  • ✓ Tool execution sandbox
  • ✓ Retry budget and loop guards
  • ✓ Memory decay and boundary enforcement

Proof of Fitness Protocol

Competing agents, objective scoring, tamper-evident chain. Goal-agnostic experimentation that works end-to-end.

  • ✓ Core protocol engine (fx-consensus crate, 74+ tests)
  • ✓ Orchestrator with cross-evaluation and safety gates
  • ✓ CLI: experiment run/chain/show/verify
  • ✓ Real LLM + cargo workspace evaluation
  • ✓ Subagent-driven nodes (full Fawx agents as experiment nodes)
  • ✓ Experiment tool callable from chat and channels
  • ✓ Post-hoc build verification with retry
  • ✓ Recursive learning (chain history feeds into prompts)
  • ✓ Auto-chain iterative refinement (--max-rounds)
  • ✓ Canonical chain hashing with legacy fallback
  • ✓ Cargo-scoped evaluation (-p target crate only)
  • ✓ Candidate patch persistence in chain
  • ✓ Sequential node execution (--sequential)
  • ✓ Diagnostics (--detail, --raw)
