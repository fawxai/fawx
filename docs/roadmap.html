Fleet Task Execution

  • ○ Worker mode (fawx serve --fleet)
  • ○ Task dispatch over Tailscale HTTP
  • ○ Heartbeat monitoring + health checks
  • ○ PAT borrowing (read-only GitHub access for workers)
    • Security note: GitHub PATs with contents:write can merge PRs, not just push branches. When implementing PAT sharing to subagents/workers:
      ① Branch protection on staging + main — require review from repo owner before merge (GitHub-level enforcement, not policy)
      ② Fine-grained PAT scoped to fawx repo only (limits blast radius)
      ③ No administration permission on the PAT (prevents subagents from weakening branch protection)
      ④ Result: subagents can push branches + create PRs + comment, but cannot merge to protected branches without owner review approval. dev stays open for automated pipeline.
  • ○ Fleet-aware experiment scheduling (--fleet flag)
