SSE Streaming Implementation

The server sends events using the standard SSE protocol. Here's the core handler:

rust
async fn stream_response(session_id: &str) -> impl IntoResponse { let (tx, rx) = mpsc::channel(32); tokio::spawn(async move { while let Some(chunk) = model.next_token().await { tx.send(format!("data: {}\n\n", chunk)).await.ok(); } }); Sse::new(ReceiverStream::new(rx)) }

Key points:

  • Each token arrives as a text_delta event
  • Tool calls emit tool_call_start → tool_call_complete → tool_result
  • Stream ends with a done event containing the full response
