Fawx Architecture

A personal agentic engine — designed around a recursive agentic loop, structured memory, strict safety kernel, and deep personalization. TUI-first. Desktop-first with GPU. Shells are replaceable.

🧠 The agent is personal. It serves one human. It learns their identity, taste, communication style, preferences, and needs. Every perception, plan, action, and judgment is shaped by who that human is — not by generic defaults.
🔒 Safety is architecture, not trust. The kernel enforces policy, permissions, budgets, and rollback. The agent operates within boundaries it cannot modify. Loadable intelligence makes the agent useful; compiled invariants make it safe.
🔄 The loop is recursive. Real tasks require decomposition, sub-loops, and evidence gathering. The agentic loop calls itself — with managed depth, scoped context, and independent checkpointing at every level.
💤 Memory is alive. The agent dreams — consolidating experiences into knowledge, strengthening what matters, pruning what doesn't. Memory isn't storage; it's a learning system that makes the agent better at serving its human over time.
🦀 Desktop-first with GPU. Everything built today — engine, memory, skills — runs unchanged as new shells appear. TUI validates the loop; Android/iOS/desktop render UISpec. No throwaway work.
